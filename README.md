# 🌸 Flower Shop

Современный интернет-магазин цветов, разработанный с использованием Vue.js и Firebase.

## 🚀 Возможности

- Регистрация пользователей
- Авторизация
- Каталог товаров
- Просмотр карточек товаров
- Адаптивный интерфейс
- Firebase Authentication
- Cloud Firestore

## 🛠️ Стек технологий

- Vue.js
- JavaScript
- HTML5
- CSS3
- Firebase
- Firestore

## ⚙️ Запуск

```bash
npm install
npm run dev
```
